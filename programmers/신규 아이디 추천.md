# 신규 아이디 추천

### 알고리즘 풀이

- 

```javascript
function solution(new_id) {
  let answer = "";

 answer = new_id
    	.toLowerCase()
        .replace(/[^a-z-_.0-9]/g,'')
        .replace(/[.]+/g,'.')
        .replace(/^[.]|[.]$/,'')
        .replace(/^$/,'a')
        .slice(0,15)
        .replace(/[.]$/,'');
        
	while(answer.length<3){
    	answer+=answer.slice(-1);
    }
  return answer;
}
```
